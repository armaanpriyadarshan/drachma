# Go-to-Market

Four phases. Each produces revenue, learning, or a defensible asset — ideally all three.

## Phase 0 — AI visibility audits (this week)

Manual outreach to DTC brand founders selling physical products. Offer: a one-time AI visibility audit.

- **What it is.** We test how every major AI platform sees the brand — ChatGPT, Claude, Perplexity, Gemini, agent-mediated shopping surfaces. We diagnose why they're invisible. We provide specific, actionable fixes.
- **Price.** $500 per audit.
- **Turnaround.** 48 hours.
- **Why this phase.** Generates revenue, builds market knowledge about what's actually broken in agent-mediated discovery, and seeds the first brand relationships we'll need for Phase 1.

## Phase 1 — Single-category API + creator pipeline

Pick one product category. Build the Drachma API and the creator attestation pipeline end to end.

- Recruit 20–30 creators for the category.
- Onboard 50–100 brands.
- Ship an MCP server so any agent developer can query the feed.
- Validate that the ranking model produces outputs that brands, creators, and agent builders actually trust.

Success criterion: an external agent developer plugs in the MCP server and ships a recommendation flow against Drachma without our help.

## Phase 2 — Close the feedback loop

Collect post-purchase outcome data from agents using the feed. Wire it into the ranking algorithm. Expand to adjacent categories.

- Outcome signals update creator reputation.
- Outcome signals refine niche-fit matching for similar user profiles.
- Each new category compounds the creator network and the rubric library.

Success criterion: measurable improvement in post-purchase satisfaction for Drachma-sourced recommendations vs. the Phase 1 baseline, and creator reputations that actually separate signal from noise.

## Phase 3 — Standard recommendation layer

The feed becomes the standard recommendation layer that agents query across categories. Network effects compound. The outcome-data moat widens as the longitudinal record grows.

- Brands pay for distribution because that's where agents are buying from.
- Creators participate because reputation is a portable credential.
- Agents integrate because the outcome data isn't replicable from public signals.

## What we are not doing

- No advertising product. Brands pay for access, never for placement.
- No consumer-facing app. Users interact via their agents, not ours.
- No influencer partnerships. Creators are evaluators, not promoters.
